# C-Statistics-Library
Biblioteca em C para cálculo de medidas estatísticas básicas: média, mediana, moda, variância, desvio padrão, mínimo, máximo e amplitude.

> ⚠️ **Atenção / Warning:**  
> Esta biblioteca ainda está em desenvolvimento e não está completa. Estamos trabalhando para adicionar mais funções estatísticas e melhorias em breve.  
>
> This library is still under development and not yet complete. We are working on adding more statistical functions and improvements soon.

## Funções

### media
Calcula a média de um array de números.
**Parâmetros:** `double numeros[]`, `int tamanho`
**Retorno:** `double` → média

### mediana
Calcula a mediana de um array de números.
**Parâmetros:** `double numeros[]`, `int tamanho`
**Retorno:** `double` → mediana

### moda
Calcula a moda de um array de números.
**Parâmetros:** `double numeros[]`, `int tamanho`
**Retorno:** `double` → moda
